## 1. 使用建议

### 1.1 运行前准备（请先完成再启动脚本）

- **展开玩家列表**：运行前请在游戏内展开玩家列表，避免脚本点击到被折叠的 UI。
- **速率调整为手动**：将「速率调整」切换到**手动模式**，与脚本滚轮纠偏、调速区域逻辑一致。
- **实验面板保持关闭**：启动前请保证**实验面板处于关闭状态**（未按 `E` 打开），避免初始画面与标定不符。
- **角色站位**：尽量将角色移到**视野稳定、遮挡较少**的位置；**推荐靠近手术台的玻璃门**一带，便于识别与点击。

### 1.2 标定与日常维护

- 首次使用请按 **[标定示例.md](标定示例.md)** 的推荐顺序（阶段 1～6，及可选点赞分步）；完整覆盖开始、敏感进度条、女/男条、恢复体力、调速、特殊动作、高潮、「再来一次/结束」等；开启实验切换时须完成实验卡片、身体部位、拉出新实验滚动、实验选定标志等；
- 开启实验切换模式前，须完成该文档阶段 1、2 及与实验相关的标定（含「拉出新实验滚动」滚动量）；
- 游戏画面设置变动较大（亮度、HDR、UI 比例）后建议重新标定；
- 遇到误识别优先缩小标定区域，减少背景干扰。

## 2. 热键说明


| 热键    | 功能                             |
| ----- | ------------------------------ |
| `F1`  | 紧急暂停 / 恢复运行                    |
| `F2`  | 在**实验切换模式**下：立即暂停，并置位「**恢复运行后优先切换下一实验**」。通常再按 `F1` 恢复后，脚本会先执行切换序列（先 `ESC` 退出当前实验；若首次 `ESC` 后 **恢复体力按钮** 仍能被模板识别，则短暂延迟后**持续按 `ESC`** 直至该按钮消失，视为回到大厅/列表侧，再继续 `ESC → 延时 → E → …` 打开面板并顺延实验卡片）。未开启实验切换时仅暂停，不产生切换。 |
| `F3`  | 切换「结束后强制整轮点赞（一次性）」（需点赞功能已开启） |
| `F11` | 重播"拉出新实验滚动"标定动作（用于调试滚动标定）      |
| `F12` | 切换所有标定区域的覆盖层显示（用于可视化标定位置）      |

## 3. 常见问题

### 3.1 为什么全屏看不到悬浮窗？

- 真独占全屏下普通窗口很难叠在最上层；
- 建议优先使用无边框全屏；
- 可尝试勾选 `全屏模式兼容`（实验）。

### 3.2 程序异常退出怎么排查？

- 查看：`logs/error.log`

### 3.3 点击"继续"表现异常？

- 目前按钮已加防抖，若仍异常可看日志并反馈触发路径。

### 3.4 实验切换模式下程序自动暂停？

- 检查是否完成了全部实验切换相关标定项（**实验选定标志**、**恢复体力按钮**、**实验卡片**、**身体部位**）；继续运行前还会校验 12 点 / 7 点是否齐全；若进行到第 12 卡需滚动**换行**（露出**下一行**四个实验），还须正确配置 **「拉出新实验滚动」** 的点位与向下滚动档位；
- 若 2 号和 5 号身体部位的实验均已用尽，程序会暂停并提示"全部实验已完成"。
