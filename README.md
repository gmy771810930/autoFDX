# autoFDX（Fallen Doll 自动化）

> 当前版本支持：模块化架构、悬浮控制台、逐项标定、自适应分辨率/比例、进度条纠偏、可选点赞策略、实验切换模式、女进度条停滞恢复、敏感度不足自动补偿、「拉出新实验滚动」可配置滚动量及 F11 重播调试。

---

## 1. 项目简介

`autoFDX` 是一个基于图像识别与鼠标自动化的辅助脚本，核心目标是：

- 在不同分辨率、缩放、显示模式下尽量稳定运行；
- 通过用户标定提升模板匹配精度；
- 用悬浮窗提供可视化控制、状态显示与热键安全暂停；
- 在流程关键阶段做进度条平衡纠偏，提升触发稳定性；
- 支持实验切换模式，自动顺序完成多个实验并按需切换。

## 2. 环境与安装

### 2.1 Python 版本

- 推荐：`Python 3.10+`（Windows 环境下运行）

### 2.2 依赖安装

推荐小白用户直接双击：

- `setup_windows.bat`

如需手动安装，在项目根目录执行：

```bash
pip install -r tools/requirements.txt
```

> 说明：`threading`、`sys`、`winsound` 为 Python 标准库，无需 `pip install`。

---

## 3. 启动方式

### 3.1 直接启动

```bash
python fallen_doll.py
```

### 3.2 双击 `.py` 启动（已支持）

- 已实现 Windows 下启动自动隐藏 CMD 控制台窗口；
- 若发生未捕获异常，会写入日志：
  - `logs/error.log`

### 3.3 一键环境安装（推荐）

- 双击 `setup_windows.bat`
- 脚本会自动：
  - 检测/安装 Python
  - 安装 `tools/requirements.txt` 中依赖

## 5. 性能与稳定性说明

已做的关键优化：

- 模板图缓存（避免重复磁盘读取）
- 标定区域优先局部截图匹配（避免频繁全屏截图）
- 进度条独立标定（女/男）降低误判
- 滚轮与主检测线程职责分离（关键点击优先）
- UI 位置持久化、展开收起高度自适应
- 指数移动平均（EMA）平滑进度条采样，抑制单帧噪声
